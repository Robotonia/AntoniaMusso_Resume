# AntoniaMusso_Resume

Thank you for checking out my resume! Here, you'll find the Quarto document and .css file to generate the resume.

This document uses the `weasyprint` pdf engine which you can download [here](https://doc.courtbouillon.org/weasyprint/stable/first_steps.html#installation "Installation instructions for WeasyPrint pdf engine").

For a tutorial on formatting PDF version of a Quarto CV, Cynthia Huang's [blog post](https://www.cynthiahqy.com/posts/cv-html-pdf/ "Link to Cynthia Huang's blog post on pdf cv in Quarto") is very helpful!
